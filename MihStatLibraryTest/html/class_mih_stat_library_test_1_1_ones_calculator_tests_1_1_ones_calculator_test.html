◆ CalculateOneFile01010101_131MBTest()

void MihStatLibraryTest.OnesCalculatorTests.OnesCalculatorTest.CalculateOneFile01010101_131MBTest ( )

Тест на вычисление количества единичных бит в файле:

  1. Вычисление количества единичных бит в файле размером 131 МБ из байт 01010101. Сравниваются полученное значение умноженное на 2 с размером файла в битах

◆ CalculateOneInBigIntegerTest()

void MihStatLibraryTest.OnesCalculatorTests.OnesCalculatorTest.CalculateOneInBigIntegerTest ( )

Тест на вычисление количества единичных бит данных из BigInteger:

  1. Сравнивается количество единичных бит в числе 987345834572187234912834 = 1101_0001_0001_0100_0001_1111_1111_1100_0011_0000_1111_1001_0111_0000_0011_0010_1100_1010_0100_0010 с 37 (посчитанно вручную)
  2. Сравнивается количество единичных бит в первых 30 битах в числа 987345834572187234912834 = 1101_0001_0001_0100_0001_1111_1111_1100_0011_0000_1111_1001_0111_0000_0011_0010_1100_1010_0100_0010 с 37 (посчитанно вручную)

◆ CalculateOneInBlockDataTest()

void MihStatLibraryTest.OnesCalculatorTests.OnesCalculatorTest.CalculateOneInBlockDataTest ( )

Тест на вычисление количества единичных бит в блоке данных:

  1. Сравнивается количество высчитанных бит в блоке данных размером 100.000.000 байт из файла, содержащего байты 00001111 с количеством байт в блоке, умноженным на 4 (количество единичных бит в каждом байте)

◆ CalculateOneInMarkTableTest()

void MihStatLibraryTest.OnesCalculatorTests.OnesCalculatorTest.CalculateOneInMarkTableTest ( )

Тест на вычисление количества единичных бит в маркировочной таблице:

  1. Сравнивается количество единичных бит в маркировочной таблице размерностью 16 (чтоб не осталось необсчитанных хвостов), посчитанной на файле, содержащем 131 МБ байт 010100101 с размером файла в битах, поделенном на 2 (в каждом байте половина бит единичные)

Тест на исключение при вычислении количества единичных бит в маркировочной таблице с разными смещением и размерностью:
